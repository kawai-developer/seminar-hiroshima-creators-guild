- レイアウト: 3カラムカード
- 見出し: 「あなたはどのタイプ？」

**スライド10 — Cタイプの対比**

- 右（マゼンタ）「でも現実は」: 議事録まとめに1時間 / 提案書の文章に半日 / 見積もりで頭が痛い
- トークスクリプト: 「伝える人は、人と話して方向を決める瞬間が一番輝く。でも現実は事務に追われている」

**スライド11 — Gタイプの対比**
- レイアウト: 2カラム比較
- 見出し: 「G — 考える人」
- 左（シアン）「本当はやりたいこと」: IA設計・構造の整理 / ユーザー体験の設計 / 全体最適の判断
- 右（マゼンタ）「でも現実は」: 情報の海で溺れる / リサーチの整理に時間 / 雑務で思考が中断
- トークスクリプト: 「考える人は、構造を考える時間こそが価値。でも散らかった情報の整理に追われている」

## レイアウトパターンの定義

スライド生成AI向けに、使用する6つのレイアウトパターンを定義します。

### 1. center（中央配置）
- 全要素を画面中央に配置
- 見出し＋任意で本文
- 最も多用されるパターン

### 2. section（セクション区切り）
- マゼンタ（`#EC008C`）ベタ背景
- 白文字で大見出し＋サブテキスト
- 各セクションの頭に1枚

### 3. top-bottom（番号付きリスト）
- 見出しが上、番号付きカードが縦に並ぶ
- 各カードは白背景、左に番号（色付き）、右にテキスト
- 3〜6項目に対応

### 4. split-2（2カラム比較）
- 見出しが上、下に2カラム
- 各カラムは白背景カード、上部に色付きボーダー
- カラム内にタイトル＋箇条書き

### 5. split-3（3カラムカード）
- 見出しが上、下に3カラム
- 各カードは白背景、中央寄せ
- カード内にラベル（丸バッジ）＋タイトル＋説明

### 6. left-right（左右テキスト分割）
- 見出しが上、左にメインテキスト、右にマゼンタ太字のアクセントテキスト
- 左6:右4の比率
