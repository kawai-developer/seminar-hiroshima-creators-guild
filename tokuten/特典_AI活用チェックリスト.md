## 自分でやる（聖域）

あなたの感性・判断・こだわりが必要な工程。ここにAIを入れる必要はない。

- [ ] 配色の最終決定
- [ ] レイアウトの構図設計
- [ ] フォントの選定・組み合わせ
- [ ] 余白のバランス調整
- [ ] ブランドのトーン＆マナー判断
- [ ] 「気持ちいい / 気持ち悪い」の直感的判断
- [ ] クライアントの本当の課題を見抜くこと
- [ ] デザインに込めるストーリーの設計

## AIに丸投げ（やりたくない作業）

感性が不要な作業。時間だけ食って、あなたの創造性を削るもの。

### 言語化・文章系

- [ ] 提案書・企画書の文章
- [ ] メール・チャットの返信下書き
- [ ] キャッチコピーの案出し（選ぶのは自分）
- [ ] ポートフォリオの紹介文

- [ ] MTG議事録のまとめ
- [ ] ヒアリング内容の要件整理
- [ ] 競合調査の情報整理
- [ ] サイトマップ・IA設計の初期案

### 素材・加工系
- [ ] 背景画像の拡張・生成
- [ ] 画像の切り抜き・背景除去

- [ ] カラーバリエーションの検討案

### 事務・管理系
- [ ] 見積書のテンプレート作成
- [ ] スケジュールの逆算

## 判断基準：迷ったときの3つの問い

| 問い | 「はい」なら |
|:---|:---|
| その作業に「自分のこだわり」が必要？ | 自分でやる |
| 他の人がやっても同じ結果になる？ | AIに任せる |
| その作業をやっているとき、楽しい？ | 自分でやる |
