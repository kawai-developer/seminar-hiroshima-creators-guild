# やりたくない工程→AI指示文 変換シート

**【AIでデザインするな】セミナー参加者限定**

> 工程名のままだと、AIは動けません。
> 「何を作ってほしいか」「何を渡せるか」に言い換えるだけで、AIは部下になります。

---

## この1枚の使い方

1. やりたくない工程を1つだけ選ぶ
2. AIに渡せる材料を書き出す
3. 欲しい成果物の形を決める
4. テンプレートに当てはめて、GeminiやChatGPTに貼る

まずは60点のたたき台が出れば成功です。
最後の判断だけ、自分でやってください。

## STEP 2. AIに渡せる材料を書く

- 案件名 / 背景: 【　　　　　　　　　　　　　　　　　　　　　　　　　　　】
- 目的: 【何のためにこれを作るか】
- 使える材料: 【メモ / ラフ / URL / 画像 / 議事録 / 箇条書き】
- 誰向けか: 【クライアント向け / 社内向け / 自分用 など】
- 欲しい成果物: 【説明文 / 表 / メール / 見出し案 / TODO一覧 など】
- 出力形式: 【3案 / 200字 / 表形式 / 箇条書き / スライド5枚分 など】
- 条件 / NG: 【丁寧に / 専門用語なし / 事実ベース / 実在企業名を出さない など】

## STEP 3. 1本のAI指示文に変える

### そのまま使える基本テンプレート

```text
あなたは【役割】です。
以下の情報をもとに、【欲しい成果物】を作成してください。

■ 背景 / 目的
【案件名や状況、何のために作るか】

■ 材料
【メモ、ラフ、議事録、URL、画像情報など】

■ 読み手
【誰に見せるものか】

■ 出力形式
【文字数、案数、表形式、箇条書きなど】

■ 条件
【トーン、含めること、避けること】
```

### 急ぐときの1行版

```text
あなたは【役割】です。【材料】をもとに、【誰向け】の【欲しい成果物】を【出力形式】で作成してください。条件は【条件】です。
```

## 役割の決め方 早見表

| やりたくない工程 | AIに渡す役割 |
|---|---|
| デザイン意図の説明 | デザインディレクター |
| 提案書の文章化 | プレゼン構成のプロ |
| 議事録・要件整理 | プロジェクトマネージャー |
| コピー案出し | コピーライター |
| 競合整理・リサーチ要約 | 編集者 / リサーチャー |
| 見積もり・スケジュール | 営業事務 / 制作進行アシスタント |

迷ったら、「その作業を得意にやる人の肩書き」を役割にしてください。

## H / C / G 別サンプル

### H: 作る人

```text
あなたはコピーライターです。このLPラフとブランドトーンをもとに、20代女性向けのヒーロー見出しを10案出してください。各案は15文字以内、上品だが気取りすぎないトーンで、なぜ刺さるか一言ずつ添えてください。
```

### C: 伝える人

```text
あなたはプレゼン構成のプロです。以下のヒアリングメモをもとに、クライアント提案用のスライド構成を6枚分作成してください。流れは「課題→解決策→この方向性にした理由→期待効果」で、各スライドに話す要点も付けてください。
```

### G: 考える人

```text
あなたはプロジェクトマネージャーです。以下のMTG議事録を、要件・未決事項・次回アクションに整理してください。出力は表形式で、曖昧な点は「要確認」と明記してください。
```

## 出力が弱かったときの追い指示

- 「もっと具体例を3つ入れて」
- 「専門用語を減らして」
- 「表形式にして」
- 「クライアント向けに丁寧な言い回しにして」
- 「結論から先に書いて」

AIは1回で当てるより、2回で整えるほうが早いです。
